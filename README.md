# shop-helper

Local CLI tool for e-commerce merchants to manage products, customers and reduce costs.

## Features

- **Batch Image Processing**: Resize product images, add watermarks, generate previews to reduce returns
- **Customer Management**: Import/export CSV data, search and filter without contact limits
- **Order Tracking**: Local SQLite storage to avoid SaaS subscription fees
